*lvimrc*   

        Local vimrc configuration files!


    This plugin will simply and automatically source
all files names ".lvimrc" between the current working 
directory (the directory vim was started in) and the
directory pointed to by the $HOME variable.  

That's all :)  enjoy!

=========================================================

:Lvimrc                                    *:Lvimrc*
            Print the list of all files loaded by this
            plugin

 vim:set et sw=4 ts=8 tw=78 ft=help:
